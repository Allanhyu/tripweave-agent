# Changelog

本项目遵循 [Semantic Versioning](https://semver.org/)。

## [1.0.0] - 2026-07-14

### Added

- 手写 ReAct Agent、工具注册和记忆管理。
- FastAPI 异步旅行规划接口。
- Vue 3 旅行规划工作台。
- 高德地图、OpenWeather 与 Tavily 工具。
- 多城市规划、预算与路线约束、天气风险重规划。
- 行李清单、穿搭建议、知识图谱和 PDF 导出。
- 开源文档、环境变量模板、基础测试和 GitHub Actions。

### Security

- 将运行时密钥、本地用户状态、日志和遗留 Cookie 工具排除出公开仓库。
